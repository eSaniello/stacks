Welcome to Stacks API

Basic use of this API

De stacks API biedt CRUD operaties voor
Routes
  • freelancers
  • klant
  • categorie
  • post

al deze routes bezitten de betreffende files voor de verschillende operaties(create/read/read_one/update/delete)

De URL voor Requests ziet er uit als volgt:
        http://stacks.serveo.net/stacks/backend/api/[route]/[operatie].php

Het creeren van niewe Freelancers

endpoint: http://stacks.serveo.net/stacks/backend/api/freelancers/create.php

        header("Access-Control-Allow-Origin: *");

        header("Access-Control-Max-Age: 3600");
        header("Access-Control-Allow-Headers: Content-Type, Access-Control-Allow-Headers, Authorization,
        X-Requested-With");
